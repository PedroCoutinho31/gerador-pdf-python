Gerador de PDF em Python

Aplicação desenvolvida em Python utilizando a biblioteca ReportLab para geração automática de PDFs personalizados.

Funcionalidades

Geração de recibo em PDF

Inserção dinâmica de nome, valor e descrição

Data automática

Estrutura adaptável para certificados, contratos e relatórios

Tecnologias utilizadas

Python 3

ReportLab

Como executar

Instale as dependências:

pip install -r requirements.txt

Execute:

python main.py
